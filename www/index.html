Firefox

You will need to enter http://some-company.net/proxy.pac into the browser Automatic proxy configuration URL to continue using the internet while inside this some-company.net network. After this, if you carry your laptop outside this network, then you may also need to roll-back to previous configuration without our intenal proxy. Detailed steps are as follow:

  1. Open Options from Tools > Options
  2. and then Network > Settings
  3. Remember or take note of your current configuration before modifying. Fill in new configuration http://some-company.net/proxy.pac by selecting the field "Automatic proxy configuration URL"
  4. You can roll-back to previous configuration in the future by finding the above Options window and re-entering your previous configuration kept earlier
  5. Go to some internet address and the authentication box will appear before you can continue. For other application to work e.g. Yahoo Messenger, Skype, etc. you'll need to configure individually or if they use system wide configuration just follow the Internet Options modification steps (check IE section)

Chrome

You will need to enter http://some-company.net/proxy.pac into the browser Use automatic configuration script to continue using the internet while inside this some-company.net network. After this, if you carry your laptop outside this network, then you may also need to roll-back to previous configuration without our intenal proxy. Detailed steps are as follow:

  1. Open Settings from Customize and control Google Chrome
  2. Search for proxy in the right box to make the Change proxy settings... button to appear
  3. Clicking the above button will take you to the system Internet Properties which is the same for Windows IE. Choose Connections > LAN Settings
  4. Remember or take note of your current configuration before modifying. Fill in new configuration http://some-company.net/proxy.pac by selecting the field "Use automatic configuration script"
  5. You can roll-back to previous configuration in the future by finding the above Settings and re-entering your previous configuration kept earlier
  6. Go to some internet address and the authentication box will appear before you can continue. For other application to work e.g. Yahoo Messenger, Skype, etc. you'll need to configure individually if they're not using the above system wide configuration.

Android

You will need to enter some-company.net and {port number} into the WiFi settings's Proxy host name/port to continue using the internet while inside this some-company.net network. Detailed steps are as follow:

  1. Once connected to our WiFi, open Modify network config. by tap-and-holding our WiFi access point name (check for the one with the status: connected). In the screenshot example tap More settings > Pi (connected)
    (screenshot below is taken Android 4.04 ICS)
  2. Scroll then active Show advanced options. Choose Proxy settings > Manual
  3. Remember or take note of your current configuration before modifying. Fill in new configuration some-company.net (Proxy host name) and {port number} (Proxy port)
  4. Go to some internet address and the authentication box will appear before you can continue. For other application to work e.g. Yahoo Messenger, Skype, etc. you'll need to configure individually if they're not using the above system wide configuration.
