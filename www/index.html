Firefox

You will need to enter http://some-company.net/proxy.pac into the browser Automatic proxy configuration URL to continue using the internet while inside this some-company.net network. After this, if you carry your laptop outside this network, then you may also need to roll-back to previous configuration without our intenal proxy. Detailed steps are as follow:

  1. Open Options from Tools > Options
  2. and then Network > Settings
  3. Remember or take note of your current configuration before modifying. Fill in new configuration http://some-company.net/proxy.pac by selecting the field "Automatic proxy configuration URL"
  4. You can roll-back to previous configuration in the future by finding the above Options window again
